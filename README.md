# fuck-venmo

Yeah, fuck 'em.

In future, this library could be generalized to "fuck-x" for many
values of X.

## owo wats dis?

It's a software library for automating the process of arguing with
Venmo customer support for months on end. This is because Venmo would
repeatedly lock me out of my account and provide absolutely no way to
resolve this other than to tell me to contact them manually every
single time this happened and go back and forth with them ten times
before they would agree to unlock it. Automation ensued. This seemed
to be their desired outcome, despite my repeatedly pointing out that
it made no sense.
